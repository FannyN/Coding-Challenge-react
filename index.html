What is React?

React is a front-end library developed by Facebook. It is currently one of the most popular JavaScript libraries and has a strong foundation and large community behind it.

React is used for handling the view layer for web and mobile apps and makes it easy to structure and reuse the code. Normally in vanilla JavaScript, when you want to re-use some code, you write it as a function. In React, you write your code as a component. You can create a class and then place multiple instances of that class wherever you want.
